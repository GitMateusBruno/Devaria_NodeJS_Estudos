ERN: Acelerador de particulas que fica na Europa. Necessidade das pessoas acessarem o mesmo documento. Criação de protocolo de redes e comunicação entre máquina

Protocolo de rede: Conjunto de regras que permitem que dispositivos se comuniquem e troquem informações entre si. Cada um com uma finalidade especifica

IP; identificador único de cada dispositivo eletrônico. DNS (domínio) – Acessar IP através de um único nome. TCP/IP: Pacotes enviados e recebidos durante a comunicação.

Protocolo padrão de comunicação cliente-servidor. Cabeçalho (metodo, segurança, cookies). Corpo (informaçao em si, JSON). Endereço (dns, ip, porta). Status de resposta (200, 404). Formato JASON; Chave : valor; String entre aspas duplas; Arrays[]; Objetos{...}

GET; POST; PUT; DELETE. 200 OK; 404 NOT FOUND; 500 SERVER ERRO; 400 BAD REQUEST; 401 Unathorized

Interface de comunicação. Permite que diferentes softwares se comuniquem. Integração entre sistemas de forma fácil.
